## Stacks

### Docker Compose

Faça o download do arquivo [docker-compose.yml](docker-compose.yml) e rode:

```bash
docker compose up -d
```

Após os serviços estarem iniciados:

* Acesse http://localhost:8080
* Crie um novo site
* Acesse http://universidade.localhost
